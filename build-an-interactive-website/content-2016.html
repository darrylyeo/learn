It is different from a regular class in that they represent a special state of an element. --- Transitioning Hovered Elements The `transition` property allows you animate certain properties of CSS elements when they change state. To use it, set its value to the name of the property you wish to animate, and a duration in seconds (`s`). ```a { color: blue; transition: color 0.3s; } a:hover { color: green; }``` To animate multiple properties at once, separate each one with a comma: ```a { background-color: black; color: blue; transition: color 0.3s, background-color 1s; } a:hover { background-color: white; color: green; }``` To animate every single possible property, use the `all` keyword: ```a { transition: all 0.3s; }``` --- Review Activity: Make a Button! {{ 1.
