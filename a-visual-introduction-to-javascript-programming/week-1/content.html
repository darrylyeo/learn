              --- The `:hover` Pseudoclass Add `:hover` to any selector to make a CSS rule apply only when the mouse is hovered over an element. ```a:hover { color: green; }``` `:hover` is an example of a *pseudoclass*. To use it, set its value to the name of the property you wish to animate, and a duration in seconds (`s`). ```a { color: blue; transition: color 0.3s; } a:hover { color: green; }``` To animate multiple properties at once, separate each one with a comma: ```a { background-color: black; color: blue; transition: color 0.3s, background-color 1s; } a:hover { background-color: white; color: green; }``` To animate every single possible property, use the `all` keyword: ```a { transition: all 0.3s; }``` --- Review Activity: Make a Button! {{ 1.
